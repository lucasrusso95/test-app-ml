# ProductImage

## Usage
You can use it in your code like a React component with the jsx tag:

```jsx
<ProductImage 
    id={id} 
    url={url}
    width={width}
    title={title}
/> 
```

| Prop name | Type | Description                       
| - | - | - | - |
| `className` | `String` | Class name
| `url` | `String` | Image url   
| `width` | `String` | Image width  
| `title` | `String` | Image title

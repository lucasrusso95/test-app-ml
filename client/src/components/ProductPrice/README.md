# ProductPrice

## Usage
You can use it in your code like a React component with the jsx tag:

```jsx
<ProductPrice 
    className={className}
    price={price}
>
    {children}
</ProductPrice>
```

| Prop name | Type | Description                       
| - | - | - | - |
| `className` | `String` | Class name
| `price` | `{ currency: String, amount: Number, decimals: Number }` | Title
| `children` | `Node` | Children
